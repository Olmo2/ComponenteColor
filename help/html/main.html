JavaHelp para Componente personalizado jTextView, cambio de color del fondo

Esta aplicación contiene un componente personalizado, un jTextField, el cual posee un atributo "pigmento" que es un obj que contiene dos objetos de tipo "Color" los cuales definen el color del fondo y del texto del mismo.

Por defecto los colores son rojo/(255,0,0) y amarillo(255, 255, 0) respectivamente, para editarlos se ha implementado un soporte de edición. se deberán introducir los código de los colores deseados en formato R,G,B , es decir "255,255,255".

El uso pensado para este componente es utilizar el envento de java "focusGained" para cabiar el color del jTextField si el usuario está sobre el.
